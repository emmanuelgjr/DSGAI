DSGAI05 · Attack Path

Data Integrity & Validation Failures

Structurally valid but semantically malicious payloads that bypass ingestion controls

Primary Attack Flow
🎯 Attacker
Crafts structurally valid but malicious payload
→
Submits to Ingestion Boundary
File upload, API feed, snapshot import, labeling queue
→
Passes Format Validation ✓
Schema check passes — semantic content unchecked
→
Corrupts Pipeline Silently
Training data, config files, or embeddings altered
→
💥 System Compromise
Backdoored model, open vector store, or exfiltrated embeddings
Two Distinct Attack Surfaces
Surface 1
Schema & Semantic Validation Bypass
Attacker crafts adversarial CSV / JSON / Parquet
↓
Payload conforms to schema — passes syntactic check ✓
↓
Contains extreme outliers, null injections, Unicode edge cases, or rare statistical patterns
↓
Silently corrupts training set or shifts feature distributions
↓
Degraded, biased, or adversarially shifted model — no alert fired
⚠ No persistent access required — exploits ingestion trust in format-valid data
Surface 2
Import & Snapshot Path Traversal
Attacker crafts poisoned snapshot archive
↓
Archive contains symlinks or path traversal sequences (e.g., ../../)
↓
Snapshot passes format validation ✓ — import triggered
↓
Path traversal writes to host service config directory
↓
Auth disabled on vector store API — full embedding index exposed
⚠ CVE-2024-3584 (Qdrant) — no auth bypass needed, only import endpoint access
Attack Vectors

① Adversarial CSV Ingestion

SCHEMA BYPASS
Attacker prepares valid-schema CSV with crafted values
↓
Passes ingestion format check ✓
↓
Extreme outliers shift label distribution in training set
↓
Model trained on corrupted data — biased or degraded behavior
↓
Analytics & evaluations derived from corrupted feature store become unreliable
⚠ Long MTTD — corruption survives format validation and appears operationally normal

② Snapshot Path Traversal (Qdrant)

ARBITRARY FILE WRITE
Attacker accesses snapshot import endpoint
↓
Crafts archive with symlinks targeting /etc/service config
↓
Archive passes format validation ✓ — import executes
↓
Modified config disables API authentication silently
↓
External attacker queries open endpoint — full embedding index exfiltrated
⚠ Service continues to operate normally — no integrity alert fires

③ Labeling Queue Injection

SEMANTIC BYPASS
Attacker injects crafted samples into labeling pipeline
↓
Samples are structurally valid — assigned correct labels
↓
Pass schema and format checks ✓ — promoted to training
↓
Rare statistical patterns embed backdoor triggers in fine-tuned model
↓
Trigger activates at inference — adversarially controlled output
⚠ No persistent data store access required — only ingestion boundary access
Resulting Impact
🗄️ Full System Compromise

📉 Silent Training Corruption
Biased or adversarially shifted model behavior
🔓 Config Tampering
Security controls removed without visible impact
⏱ Long Mean-Time-to-Detection
Corruption designed to appear operationally normal
