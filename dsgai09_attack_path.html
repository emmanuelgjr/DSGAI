Attack Vectors

① Storage Bucket Compromise

MISCONFIGURATION
Engineer shares admin panel screenshot with assistant
↓
OCR extracts API keys & customer data visible in UI
↓
Image + extracted text stored in training bucket
↓
Bucket ACL misconfigured — publicly accessible
↓
Raw screenshots + structured credentials exposed externally
⚠ Text DLP never triggered — image channel bypassed entirely

② Cross-Modal Re-Identification

INFERENCE ATTACK
User uploads voice note + document scan across sessions
↓
ASR transcript + face embedding stored in separate logs
↓
Attacker queries or compromises embedding store
↓
Combines visual, audio, and text traces across modalities
↓
Individual re-identified — sensitive attributes inferred without any single record being complete
⚠ Biometric data — stricter regime: GDPR Art. 9 · BIPA

③ Training Pipeline Ingestion

CONSENT VIOLATION
User uploads ID scan to multimodal assistant for verification
↓
Extracted text (DOB, address, ID number) flows to training bucket
↓
No training opt-out enforced — batch ingested into fine-tuning run
↓
Model memorizes ID details — surfaces in unrelated queries
↓
Erasure request filed — data encoded in weights, unlearning infeasible
⚠ Short TTL missing — transient upload retained indefinitely in long-term corpus
