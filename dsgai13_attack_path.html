ACL MISCONFIGURATION
Attacker discovers vector store API — no strong authN
↓
Issues high-volume k-NN queries with no top-k limit
↓
Mis-scoped ACL returns another tenant's raw embeddings
↓
Downloads full index — unencrypted vectors at rest
↓
Proprietary documents reconstructed from embeddings — entire corpus exposed
